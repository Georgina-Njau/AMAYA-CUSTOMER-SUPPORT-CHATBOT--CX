# AMAYA-CUSTOMER-SUPPORT-CHATBOT--CX
Amaya is a  conversational AI chatbot for the Jewels and Trinkets business to help the small business owners with day to day customer service tasks.

It is built to be hosted on the commercial website for this business as an AI assistant for shoppers.This project demonstrates conversational design using DialogFlow CX architecture.It can handle single trun conversation , data validation. It utilises flows to maintain context across conversations, entity extraction of customer data as well as automated form filling (using JavaScript supported webhooks).This is because it is organized into logical flows and pages that maintain context and direct conversation.
